# Data Setup

1. Go to:
   `https://drive.google.com/file/d/19YgknnWv4s2HSyMqMTt-6cNWhvbgpAFu/view`
2. Download the zip file.
3. Unzip it locally.
4. Copy the files from the zip's `data/` folder into this repository's `data/` folder (this folder).

Notes:
- The project expects parquet files in this directory.
- `data/*.parquet` is gitignored, so local data files will not be committed.
